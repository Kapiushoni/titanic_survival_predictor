# Проект "Прогнозування виживання пасажирів 'Титаніка'"

Це класична задача з Data Science, яка демонструє повний цикл роботи з "брудними" даними: від очищення до побудови моделі.

**Мета:** Побудувати модель машинного навчання, що прогнозує виживання пасажира на основі його даних (клас, стать, вік тощо).

### Технічний стек

* **Python**
* **Pandas:** Для маніпуляцій з даними.
* **Seaborn / Matplotlib:** Для аналізу та візуалізації даних (EDA).
* **Scikit-learn:** Для підготовки даних (imputation, encoding) та побудови моделі.

### Етапи проекту

1.  **Завантаження та Аналіз (EDA):** Дані завантажені з бібліотеки `seaborn`. Проведено аналіз пропущених значень (виявлено пропуски у `age` та `embarked`).
2.  **Візуалізація:** Побудовані графіки, що показують високу кореляцію виживання зі статтю (`sex`) та класом каюти (`pclass`).
3.  **Feature Engineering (Обробка даних):**
    * Пропущений вік (`age`) заповнено **медіанним** значенням.
    * Пропущений порт посадки (`embarked`) заповнено **модою** (найчастішим значенням).
    * Категоріальні дані (`sex`, `embarked`) перетворено на числові (Label & One-Hot Encoding).
4.  **Побудова моделі:** Дані розділено на тренувальну та тестову вибірки.
5.  **Тренування:** Натреновано модель **`RandomForestClassifier`**.
6.  **Оцінка:** Модель досягла **~80% точності** на тестовій вибірці.

### Результати

У файлі `titanic_survival_predictor.ipynb` міститься повний код, що демонструє всі кроки: аналіз, очищення даних, кодування ознак, тренування та оцінку моделі (включаючи матрицю помилок).
